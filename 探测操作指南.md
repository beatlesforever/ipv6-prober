## 服务器端：发送探测

SSH 登录云服务器后执行：

```bash
cd ~/ipv6-prober/ipv6_probe_tool

# === 全部 6 种探测一次性执行 ===

sudo .venv/bin/python3 main.py --target 2402:f000:4:1008:809:9845:ddb9:8a16 --probe-type normal --count 3 --output results/all_tests.csv && \
sudo .venv/bin/python3 main.py --target 2402:f000:4:1008:809:9845:ddb9:8a16 --probe-type spoofed-src --spoofed-src 2001:db8:dead::1 --count 3 --output results/all_tests.csv --append && \
sudo .venv/bin/python3 main.py --target 2402:f000:4:1008:809:9845:ddb9:8a16 --probe-type ext-chain --chain-len 1 --count 3 --output results/all_tests.csv --append && \
sudo .venv/bin/python3 main.py --target 2402:f000:4:1008:809:9845:ddb9:8a16 --probe-type ext-chain --chain-len 3 --count 3 --output results/all_tests.csv --append && \
sudo .venv/bin/python3 main.py --target 2402:f000:4:1008:809:9845:ddb9:8a16 --probe-type ext-chain --chain-len 5 --count 3 --output results/all_tests.csv --append && \
sudo .venv/bin/python3 main.py --target 2402:f000:4:1008:809:9845:ddb9:8a16 --probe-type ext-chain --chain-len 8 --count 3 --output results/all_tests.csv --append && \
sudo .venv/bin/python3 main.py --target 2402:f000:4:1008:809:9845:ddb9:8a16 --probe-type fragment --count 3 --output results/all_tests.csv --append && \
sudo .venv/bin/python3 main.py --target 2402:f000:4:1008:809:9845:ddb9:8a16 --probe-type routing --count 3 --output results/all_tests.csv --append && \
sudo .venv/bin/python3 main.py --target 2402:f000:4:1008:809:9845:ddb9:8a16 --probe-type abnormal-order --count 3 --output results/all_tests.csv --append

# 查看结果
cat results/all_tests.csv
```

## Windows 端：验证是否收到包

以下命令在 Windows 本机**以管理员身份运行 PowerShell** 执行。

### 1. 实时抓包统计（执行后保持运行，按 Ctrl+C 停止）

```powershell
# 监听来自服务器的所有 IPv6 报文
pktmon filter add -i 2603:c028:4505:b142:0:7859:7328:a163
pktmon start --capture --pkt-size 0 --log-type realtime
```

> 收到报文会在窗口实时滚动显示。按 Ctrl+C 停止后执行 `pktmon stop` 清理。

### 2. 快速计数：收到多少个来自服务器的 IPv6 包（运行探测后执行）

```powershell
pktmon filter add -i 2603:c028:4505:b142:0:7859:7328:a163
pktmon start --capture --pkt-size 0
# 等待 30 秒（期间运行服务器探测命令）
Start-Sleep -Seconds 30
pktmon stop
$log = pktmon etl2txt | Select-String "2603:c028"
Write-Host "收到来自服务器的包数: $($log.Count)"
```

### 3. 用 netsh 检查 ICMPv6 统计

```powershell
# 查看当前 ICMPv6 收发统计
netsh interface ipv6 show ipstats | Select-String -Pattern "Received|Sent|ICMP|Echo"
```

### 4. 用 ping6 先确认双向可达性

```powershell
# Windows 主动 ping 服务器
ping -6 2603:c028:4505:b142:0:7859:7328:a163
```

```bash
# 服务器 ping 本机（SSH 到服务器执行）
ping6 -c 2 2402:f000:4:1008:809:9845:ddb9:8a16
```

## 探测类型速查

| 探测类型 | 说明 |
|----------|------|
| `normal` | 普通 ICMPv6 Echo Request |
| `spoofed-src` | 伪造源地址 |
| `ext-chain` | 多层扩展头链 (chain-len: 1/3/5/8) |
| `fragment` | 真正 IPv6 分片 |
| `routing` | 路由扩展头 (Type 0, segleft=1) |
| `abnormal-order` | 异常扩展头顺序 (DestOpt 在 HopByHop 之前) |
