### 2. Windows 防火墙放行 ICMPv6 Parameter Problem

部分探测类型 (如 routing, abnormal-order) 可能触发目标返回 Parameter Problem. 需要先放行出站规则:

```powershell
New-NetFirewallRule -DisplayName "IPv6-Diag-Out-ICMPv6-ParamProblem" `
    -Direction Outbound -Protocol ICMPv6 -IcmpType 4 `
    -RemoteAddress 2603:c028:4505:b142:0:7859:7328:a163 -Action Allow -Profile Any
```

### 3. Windows 抓包环境初始化

每次测试前先停止旧会话, 然后重置并添加筛选器:

```powershell
pktmon stop
pktmon reset
pktmon filter add -i 2603:c028:4505:b142:0:7859:7328:a163
```
